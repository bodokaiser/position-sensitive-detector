# Position-sensitive detector

Position-sensitive detector using the Hamamatsu S5991-01 position-sensitive photodiode.

| Parameter | Value |
| --------- | ----- |
| Spatial resolution | 2 um |
| Bandwidth | 1400 kHz |
| Input voltage | +- 15 V |
| Output voltage | +- 12 V |
